## 策略含义

- 2月的 b2 PASS top3 是能打的：胜率过半、均值和中位数都为正。好消息，真是好消息。
- 3月的 b2 PASS top3 明显失效：胜率 33.33%，均值 -4.42%，中位数 -4.69%。这说明 3月不是“选得不够多”的问题，而是市场环境与 b2 PASS 排名逻辑不匹配。
- 4月的 b2 PASS top3 又重新变强：有效样本胜率 72.92%，均值 6.18%，中位数 3.46%。如果说 3 月像刀钝了，4 月就是重新磨开刃了。
- 次日开盘缺口并不能解释全部收益差异：1月平均开盘缺口接近 0，2月也接近 0，但 5日表现差异很大；真正的关键仍在后续 5日承接与趋势延续。
- 4月虽然整体偏强，但开盘端并没有同步变成明显高开：有效样本里低开仍略多于高开（25 比 21）。这说明 4月赚钱不是靠次日一致性抢筹，而是靠后续 5 日延续。
- 对交易使用来说，b2 的 PASS top3 不能跨月份无脑套用。强势环境里它像一把好刀；弱势/切换环境里它会变钝，甚至反向伤人。大家都知道，regime 是最大的老板。
