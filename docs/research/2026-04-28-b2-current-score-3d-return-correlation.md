# B2 当前评分与3日收益相关性复盘（2026-04-13 ~ 2026-04-24）

## 背景

本复盘记录在当前代码实现下，`stock-select b2` 候选样本的小分、MACD 百分制评分与未来 3 日收益之间的关系。

本次分析紧接一次 MACD 状态机与 b2 review 逻辑调整：

- MACD 使用双周期浪型状态机与百分制共振评分，再映射回 `macd_phase` 1~5。
- `classify_weekly_macd_trend()` 修复了 `W-FRI` 重采样在周中 `pick_date` 混入未来周五标签的问题。
- `b2` baseline review 不再使用 MACD verdict gate 强制改写 verdict；MACD 仍通过 `macd_phase` 参与总分与注释。
- b2 `price_position` 使用“120日高点到入选日之间最低收盘价”衡量高点回踩后的相对位置，而不是直接使用入选日收盘价。

## 小分相关性

```text
|因子|Pearson|Spearman|解读|
|---|---:|---:|---|
|total_score|-0.058|-0.099|总分与收益略负相关，当前加权不适合直接排序|
|trend_structure|+0.046|+0.077|弱正相关|
|price_position|-0.168|-0.194|明显负相关，当前位置分越高收益反而偏弱|
|volume_behavior|+0.006|+0.026|几乎无相关|
|previous_abnormal_move|-0.125|-0.195|明显负相关，高分偏弱|
|macd_phase|+0.097|+0.067|弱正相关，是小分里相对有用的一项|
|macd_percent|+0.097|+0.067|同 macd_phase|
|weekly_total|+0.056|+0.062|弱正相关|
|daily_total|-0.012|-0.019|基本无效|
|resonance_total|+0.102|+0.048|弱正相关|
|setup_bonus|+0.135|-0.024|线性正相关但排序不稳，受少数大涨样本影响|
```

核心结论：

1. 当前 `total_score` 不适合直接作为 3 日收益排序因子。
2. MACD 方向相对正确但解释力弱，需要和高位/量能/风险结构叠加使用。
3. `price_position` 与 `previous_abnormal_move` 在当前样本中呈明显负相关，需要重新审视打分方向或权重。
4. `volume_behavior` 近乎无相关，但分档表现显示 3 分档优于 4/5 分档，说明“更高量能分”不是单调更优。

## 策略含义

建议后续优化方向：

1. 降低 `price_position` 权重，或重写其高低位评分方向。
2. `previous_abnormal_move=5` 不应继续线性奖励，可考虑改为中性或轻微扣分。
3. 提高 `signal_type=trend_start` 的影响；对 `distribution_risk` 更强降权。
4. `MACD% > 90` 不应自动视为优势，需与高位/量能风险联动，否则容易纳入 `300559.SZ`、`300834.SZ` 这类高 MACD 但短线大跌样本。
5. 当前 `WATCH` 组显著优于 `PASS`，PASS 规则需要重新校准。

## 备注

本复盘为当前实现的截面回测，不代表稳定长期结论。样本仍受 2026-04-13 ~ 2026-04-24 这段市场环境影响。后续调整权重前，建议追加更多日期窗口并做滚动验证。
